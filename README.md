# 视频素材清理助手 (VideoCleaner)

一款专为短视频创作者设计的手机 App，自动识别并清理已发布到视频号的原始素材，节省手机存储空间。

## 📱 产品定位

- **目标用户**：短视频创作者、微信视频号活跃发布者
- **核心价值**：自动清理已发布素材，节省手机存储
- **支持平台**：iOS + Android (Flutter 跨平台)
- **变现模式**：广告 + 会员 (二期)

## 🎯 核心功能

| 功能 | 说明 | 状态 |
|------|------|------|
| 视频号绑定 | 授权获取发布记录 | 待开发 |
| 素材识别 | 匹配本地素材与已发布视频 | 待开发 |
| 清理策略 | 手动/半自动/全自动可配置 | 待开发 |
| 本地清理 | 清理手机相册原始素材 | 待开发 |
| 回收站 | 7 天内可恢复 | 待开发 |
| 清理记录 | 追溯每次清理详情 | 待开发 |

## 🛠️ 技术栈

- **前端框架**：Flutter 3.x
- **状态管理**：Provider / Riverpod
- **本地存储**：Hive + SQLite
- **相册访问**：photo_manager
- **通知服务**：flutter_local_notifications
- **广告 SDK**：穿山甲/优量汇 (二期)

## 🚀 快速开始


## ⚠️ 已知风险

1. **视频号 API 限制** - 微信未开放官方 API，需采用手动标记或无障碍服务
2. **iOS 后台限制** - 全自动模式在 iOS 上可能无法实现
3. **应用商店审核** - 清理类 App 审核较严格

详细风险分析见 `docs/DEVELOPMENT_PLAN.md`
