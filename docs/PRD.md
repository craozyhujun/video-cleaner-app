# 视频素材清理助手 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
**视频素材清理助手** (VideoCleaner)

### 1.2 产品定位
一款专为短视频创作者设计的手机 App，自动识别并清理已发布到微信视频号的原始拍摄素材，节省手机存储空间。

### 1.3 目标用户
- 短视频创作者（个人/小团队）
- 微信视频号活跃发布者
- 手机存储空间紧张的用户

### 1.4 核心价值
- ✅ 自动识别已发布素材
- ✅ 可配置的清理策略
- ✅ 跨平台支持（iOS + Android）
- ✅ 安全可靠（清理前确认/备份）

---

## 2. 功能需求

### 2.1 核心功能（MVP）

| 功能模块 | 功能描述 | 优先级 |
|----------|----------|--------|
| 视频号绑定 | 授权登录微信视频号，获取发布记录 | P0 |
| 素材识别 | 自动匹配本地素材与已发布视频 | P0 |
| 清理策略配置 | 用户可配置清理模式（手动/半自动/全自动） | P0 |
| 本地素材清理 | 清理手机相册中的原始素材 | P0 |
| 清理记录 | 记录每次清理的文件列表、释放空间 | P0 |
| 回收站机制 | 清理后文件进入回收站，支持 7 天内恢复 | P0 |

### 2.2 清理模式（可配置）

| 模式 | 说明 | 用户干预 |
|------|------|----------|
| 手动模式 | 用户主动触发清理 | 每次清理需确认 |
| 半自动模式 | 检测发布成功后推送通知，用户确认清理 | 每次清理需确认 |
| 全自动模式 | 后台监听发布成功，自动清理 | 无需干预（可设置白名单） |

### 2.3 配置项

| 配置项 | 可选值 | 默认值 |
|--------|--------|--------|
| 清理模式 | 手动/半自动/全自动 | 手动 |
| 回收站保留天数 | 1/3/7/15/30 天 | 7 天 |
| 清理前备份 | 是/否 | 否 |
| 备份位置 | 本地文件夹/云存储 | 本地文件夹 |
| 白名单相册 | 可选择不清理的相册 | 无 |
| 最小视频时长 | 忽略短于 X 秒的视频 | 0 秒 |
| 清理通知 | 开启/关闭 | 开启 |

### 2.4 二期功能（迭代）

| 功能模块 | 功能描述 | 优先级 |
|----------|----------|--------|
| 云存储备份 | 清理前自动备份到云端（阿里云/腾讯云） | P1 |
| 多平台支持 | 支持抖音、快手、B 站等平台 | P1 |
| 广告变现 | 接入穿山甲/优量汇广告 SDK | P1 |
| 会员体系 | 付费解锁高级功能（无限回收站、云备份等） | P1 |
| 批量管理 | 批量选择、批量清理 | P1 |
| 存储分析 | 可视化展示存储空间占用 | P1 |

## 3. 技术约束

### 3.1 平台限制

#### iOS 限制
- iOS 沙盒机制限制，无法直接访问系统相册
- 需使用 Photos Framework，用户授权后才能访问
- 后台监听能力有限，全自动模式可能受限
- 必须上架 App Store，审核较严格

#### Android 限制
- Android 10+ 分区存储限制
- 需申请读写权限
- 后台监听相对灵活
- 可上架多个渠道（Google Play + 国内应用商店）

### 3.2 微信视频号 API
- ⚠️ **风险点**：微信视频号官方 API 开放程度有限
- 可能需要通过以下方式实现：
  - 方案 A：用户手动标记已发布视频
  - 方案 B：通过无障碍服务监听发布动作（Android）
  - 方案 C：截图/录屏识别发布成功页面（需优化）

## 4. 用户流程

### 4.1 首次使用流程
```
启动 App → 授权相册访问 → 授权视频号访问 → 配置清理策略 → 完成
```

### 4.2 清理流程（手动模式）
```
打开 App → 查看可清理素材列表 → 选择要清理的文件 → 确认清理 → 完成
```

### 4.3 清理流程（全自动模式）
```
后台监听 → 检测到发布成功 → 自动匹配素材 → 移动到回收站 → 通知用户
```

---

## 5. 安全与隐私

### 5.1 数据安全
- 所有操作本地执行，不上传用户素材
- 回收站机制防止误删
- 清理记录可追溯

### 5.2 隐私保护
- 明确告知用户权限用途
- 不收集用户视频内容
- 符合 GDPR/个人信息保护法要求

## 7. 风险与应对

| 风险 | 影响 | 应对方案 |
|------|------|----------|
| 视频号 API 限制 | 无法自动识别发布状态 | 提供手动标记功能作为备选 |
| iOS 后台限制 | 全自动模式无法实现 | iOS 仅支持手动/半自动模式 |
| 用户误删 | 用户数据丢失 | 回收站 7 天保护 + 清理前确认 |
| 应用商店审核 | 上架被拒 | 提前准备审核材料，合规设计 |
